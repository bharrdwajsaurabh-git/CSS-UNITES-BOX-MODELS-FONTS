# CSS CSS-UNITES-BOX-MODELS-FONTS Demonstration

## Overview
This project is a simple demonstration of how  **CSS units, box models and fonts**   work to and style specific HTML elements. It includes examples of padding margin vh vw box-sizing and different font properties.

## Files Included
* `index.html`: The main HTML structure containing a div which contain a image and a button.
* `style.css`: The stylesheet that applies specific rules to the HTML using font properties different css units and box model.

## How to Use
1. Ensure both `index.html` and `style.css` are saved in the same directory.
2. Open `index.html` in any modern web browser to see the CSS styles applied to the HTML structure in action.
